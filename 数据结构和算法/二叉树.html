二叉树 一、定义 树是用来模拟具有树状结构性质的数据集合，根据他的特性可分为多种种类。 二叉树是一种典型的树状结构，二叉树是每个节点最多有两个子树的树结构，通常子树被称为“左子树”和“右子树”。 二、特点 1、第n层的节点数最多为2^n个节点。 2、n层二叉树的总节点数为2^(n+1)-1。 3、第一个非叶子节点为length/2。 4、一个节点的孩子节点：2n、2n+1。 三、二叉树遍历 先序遍历：遍历到该节点后，即刻输出该节点的值，再继续遍历其左子树，最后遍历其右子树。 中序遍历：遍历到该节点后，先遍历其左子树，后输出该节点的值，最后遍历其右子树。 后序遍历：遍历到该节点后，先遍历其左子树，后遍历其右子树，最后输出该节点的值。 前序遍历：根节点 + 左子树前序遍历 + 右子树前序遍历 中序遍历：左子树中序遍历 + 根节点 + 右字数中序遍历 后序遍历：左子树后序遍历 + 右子树后序遍历 + 根节点 四、对称二叉树 对称二叉树的特定 两个根结点相等。 左子树的右节点和右子树的左节点相同。 右子树的左节点和左子树的右节点相同。 node1.value === node2.value node1.left === node2.right node1.right === node2.left 五、二叉搜素树 定义：左节点比根节点小的，右节点比根节点大的。 六、二叉树的最大深度 二叉树的深度为根节点到最远叶子节点的最长路径上的节点数 思路： 一颗二叉树的最大深度等于左子树深度和右子树深度的最大值+1 function TreeDepth(root) { return !root ? 0 : Math.max(TreeDepth(root.left), TreeDepth(root.right)) + 1; } 七、二叉树的最小深度 最小深度是从根节点到最近叶子节点的最短路径上的节点数量。 思路： 左右子树都不为空，左子树最小深度和右子树最小深度的最小值+1； 左子树为空，右子树最小深度+1 右子树为空，左子树最小深度+1； function minDepth(root) { if(!root) { return 0; } if(!root.left) { return minDepth(root.right) + 1; } if(!root.right) { return minDepth(root.left) + 1; } return Math.min(minDepth(root.left), minDepth(root.right)) + 1; } 八、平衡二叉树：每个子树的深度之差不超过1
