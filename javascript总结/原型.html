javascript 原型 1. 定义 原型[[protype]]： 每个对象的拥有的一个属性，该属性指向了该对象对应的原型对象。 原型对象: 每个构造函数(其实就是每个函数)都拥有的一个属性并且其值是一个对象，用来实现原型继承。也就是使用构造函数生成的实例对象的[[prototype]]指向的值 构造函数: 用来初始化新创建的对象的函数就是构造函数，也就是使用new操作符实例化了的那个函数。 constructor: 原型对象有一个constructor属性，指向了该原型对象对应的构造函数。因为实例对象继承了原型对象的属性，所以实例对象也拥有constructor属性。 实例对象: 通过构造函数的new操作符创建的对象是实例对象，一个构造函数可以创建多个实例对象。 原型链: javascript的对象都关联了它的原型对象，而它的原型对象本身也是一个对象，也有自己的原型对象，这样层层上溯，形成了类似链表的结构，也就形成了原型链。 所有原型链的终点都是Object.prototype, 而其原型是null,null没有原型。 new: new操作符创建实例对象的过程。 1. 创建一个新的对象。 2. 这个新对象关联原型链。 3. 这个新对象会绑定到函数调用的this。 4. 如果构造函数没有返回其他对象，则自动返回这个新对象。 2. 图解原型链 https://juejin.im/post/5b729c24f265da280f3ad010 https://user-gold-cdn.xitu.io/2018/10/22/1669b91eafbae812?imageView2/0/w/1280/h/960/format/webp/ignore-error/1 1. 实例对象obj与构造函数的关系 2. 函数与Function的关系
