CSS 1. 定义 Cascading Style Sheets 层叠样式表，级联样式表。简称样式表。 2. 作用 设置HMTL网页元素的样式。 3. HTML与CSS的关系 HTML： 负责内容的展示 CSS： 负责内容元素的修饰 4. CSS的语法规范 1. 内联样式，又为行内样式： 将css样式定义在HTML元素上。 2. 内部样式，在网页的head部添加 3. 外部样式引入 5. CSS的样式特征 1. 继承性，子元素可以继承父元素的样式 2. 层叠性，可以为一个元素定义多个样式。 3. 优先级 浏览器默认设置 -> 内部样式和外部样式 -> 内联样式 4. 调整优先级: p { color: red !important; 设置优先使用该样式 } 6. CSS基础选择器 1. 选择器的作用：规范页面中的哪些元素能够使用定义好的样式。 2. 通用选择器 * { font-size: 12px; } 3. 元素选择器 div {} p{} 4. 类选择器 作用：通过元素的class属性进行引用 1.类名是自定义的，但是注意类名不能以数字开头 2.类名不能包含特殊符号（&,^,%,$,#,@） 3.可以包含（_,-） .class-name { color: red; } 5. id选择器 #id { color: red; } 6. 后代选择器：通过元素的后代关系匹配元素 .father .div { color: red; } 7. 子代选择器：通过元素的子代关系匹配（只对儿子有效） .father>.child { color: red } 8. 伪类选择器：匹配元素不用状态的选择器 :link 匹配元素未访问的状态 :visited 匹配元素访问过的状态 :hover 匹配鼠标悬停在元素上的状态 :active 匹配元素被激活的状态 :focus 匹配元素获取焦点时的状态 9. 选择器权重 标识当前选择器的重要程度，权值越大优先级越高。 元素选择器： 1 类选择器：10 伪类选择器：10 ID选择器：100 内联样式：1000 选择器的权值加到一起，大的优先 7. 尺寸属性 1. 作用：
