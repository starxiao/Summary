一、两列布局 1. 左列定宽，右列自适应 使用float和margin
.left { float: left; width: 100px; height: 100px; background: red; } .right { height: 100px; margin-left: 100px; background: green; } 使用position:absolute .left { position: absolute; left: 0; top: 0; width: 100px; height: 100px; background: red; } .right { position: absolute; right: 0; top: 0; left: 100px; height: 100px; background: green; } 使用flex .parent { display: flex; } .left { width: 100px; height: 100px; background: red; } .right { height: 100px; flex: 1; background: green; } 2. 左侧自适应，右侧固定 .left { width: 100%; height: 100px; float: left; margin-left: -100px; } .right { width: 100px; height: 100px; float: right; }
.left { overflow: hidden; height: 100px; } .right { width: 100px; height: 100px; float: right; } 3. 一列不定，一列自适应 .left { float: left; height: 100px; } .right { overflow: hidden; height: 100px; } .parent { display: flex; } .left { height: 100px; } .right { height: 100px; flex: 1; }
